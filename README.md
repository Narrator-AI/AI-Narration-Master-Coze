# AI电影解说 - 一站式爆款影视解说 Coze 模版

![Coze](https://img.shields.io/badge/Coze-Template-blue) ![MySQL](https://img.shields.io/badge/Database-MySQL-orange) ![Commercial](https://img.shields.io/badge/License-OEM%2FResale-green)

## 🌟 项目介绍

**AI电影解说** 是一套能够 **一站式自动生成完整爆款影视解说视频** 的商业级工作流模板。

这不仅仅是一个自动写文案的工具，而是将影视解说的整个生产链条全部 AI 化、自动化，真正做到 **从零到成片的一键生成**：

- ✍️ **自动生成爆款级电影解说文案**（支持模仿平台热门风格）
- 🎙️ **AI 配音**（多声音、多风格、人声级自然度）
- 🎞️ **智能画面匹配**（自动抓取镜头素材、节奏对齐、智能剪辑）
- 🎵 **BGM 自动适配**（情绪识别 + 背景音乐匹配）
- ✍️ **自动生成字幕**（多语言、可定制样式）

最终产出的是：

**一条可直接发布到抖音 / 快手 / B 站 / YouTube 的完整影视解说视频成片。**

### 👥 适用人群

- 想要 **批量生产电影/短剧解说内容** 的自媒体团队
- 想搭建 **独立 SaaS 工具并对外收费** 的创业者
- 想通过 **AI 工作流提升运营效率** 的从业者

无论是自用、私有化部署，还是商业化输出，这套模板都能提供强大、稳定、可扩展的一站式影视解说解决方案。

## 👨‍💻 核心权益与功能

本模版专为 **爆款内容批量生产** 设计，具备以下核心能力：

| 模块 | 功能描述 |
| :--- | :--- |
| **🎬 内容批量生产** | 自动化生成高质量电影 / 短剧解说文案，大幅降低生产成本，实现规模化创作。 |
| **🌀 多类型解说模式** | 支持电影解说、短剧解说、第一人称电影解说、多语种电影解说等多类型场景，满足不同平台需求。 |

## 🔧 使用说明 (部署指南)

本模版依赖 **MySQL 数据库** 来保存用户数据和 Key 管理。请务必按照以下步骤操作，否则应用无法运行。

> #### Before installing Dify，make sure your machine meets the following minimum system requirements：
> - FPU >= 2 Core
> - RAM >= 4 GiB

### 第一步：获取与复制模版

1. 登录 [Coze 官网](https://www.coze.cn/template/project/7573615119331213366?)，在左侧“模板商店”搜索关键词 **“AI电影解说”**。
    <br>
    <img width="100%" alt="第一步" src="https://github.com/user-attachments/assets/82e005be-f47a-431c-a878-ca3ae3a5f1ec" />
    <br>

2. 点击 **[复制]** 按钮，选择您的工作空间，并将其重命名（例如：`AI电影解说(自用)`）。
    <br>
    <img width="100%" alt="点击复制" src="https://github.com/user-attachments/assets/67244f56-a39f-4bd9-8487-c29b12414c67" />
    <br>

### 第二步：配置数据库 (⚠️ 核心步骤)

为了让系统正常运行并归你所有，需两步简单配置：

* **1. 连接数据库**：
    * 本系统依赖 MySQL 存储用户数据。
    * 请下载仓库内的 `coze_database_script.sql` 脚本，导入你的数据库。
    * 在 Coze 的“数据”栏目中，填入你的数据库账号密码即可。
    <br>
    <img width="100%" alt="数据库配置" src="https://github.com/user-attachments/assets/1b04926f-a616-4a87-9875-31b147f75007" />
    <br>

* **2. 品牌自定义 (OEM)**：
    * 在“用户界面”中，将所有客服/收款二维码 **替换为你自己的图片**。
    * 修改应用名称和 Logo，打造你的专属品牌。
    <br>
    <img width="100%" alt="素材替换" src="https://github.com/user-attachments/assets/1bde623d-58bb-4f76-897f-b857a0f62f85" />
    <br>

### 第三步：发布运营

配置完成后，点击 **[发布]** 至扣子商店或微信/抖音小程序，即可开始你的商业化运营。

<img width="100%" alt="发布" src="https://github.com/user-attachments/assets/6d6d5a6a-8bb1-41d2-8586-169e39cfc46f" />
<br>

1. **发布**：点击右上角“发布”，选择发布到 **“扣子商店”**，分类推荐选择 **“效率工具”**。
2. **审核**：等待平台审核通过后，用户即可在社区搜索并使用您的应用。
